About

SiteXplorer is a very flexible and small PHP and Javascript based file management system for websites with many features. It allows website admins to manage the files on a website as if using Windows Explorer. Features include:
  • Multiple views: Details, Icons, Thumbnails, Filmstrip
  • Upload multiple files, Upload/extract Zip-archives
  • Download of whole directory structures (zipped)
  • Cut/Copy/Paste Files
  • Completely keyboard-manageable, usable even without the mouse
  • Views / handling as close to Windows Exporer as possible

Installation

To install SiteXplorer, extract the archive to a directory of your choice on your PHP-enabled webserver and point your browser to index.php. During first startup, you will have to configure SiteXplorer. Configuration options are pretty straightforward and self-explanatory. If you're lost, leave the default.
